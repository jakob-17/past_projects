Path Finder


This is a typical project that I'm sure most programmers have had experience with before. During the Fall semester of my sophomore year at the U of U, we were at one point tasked with creating an algorithm in Java which successfully traversed .txt file mazes composed of X's and dimensional data.

Just as with the rest of the assignments for this class, our program was written in Java using Eclipse as the IDE.


Certainly not the most challenging of the assignments my programming partner and I had that semester, but one of my favorites nonetheless. The visual feedback that was generated by the code made the completion of the task much more satisfying.
