Lights Out


This was the first major programming project I was assigned during my time at the University of Utah. CS 1410 served as a introduction to object-oriented programming, with Java being the language of choice.

The Lights Out assignment tasked us with recreating the popular browser game of the same name. We had to complete the task in Eclipse, just as with the rest of the assignments from that course, and we were also made to learn about Java's implementation of GUIs. My two class system had a LightsOut class which handled the GUI and ActionListeners, in addition to a LightButton class that established all of the interactable "lights" on the GUI panel.

We were also to add a unique element to our game which brought some life to the user experience. The game winning screen was my answer for that, making it so each light randomly chose a color to switch to, as well as adding the You Win! text to the center light button. What made the project so enjoyable was that the entire program was by my design. There was no skeleton to build off of, or special concept which had to be included. This allowance for real creativity, coupled with the visual element of the GUI, easily makes this my favorite programming assignment from 1410.
